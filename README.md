## 材料  
(1) Arduino Nano x 1  
(2) Dfplayer MP3播放模組 x 1  
(3) 帶LED的按鈕開關 x 6  
(4) 220歐姆 x 6  
(5) 1K歐姆 x 1  
(6) 2K歐姆 x 1 (建議用0.5W以上)  
(7) 10K可變電阻 x 1  
(8) 鋰電池充電板(附帶升壓功能) x 1  
(9) 船型開關 x 1  
(10) 1000uF電解電容 x 1  
(11) micro usb 轉接板 x 1  
(12) 外盒  
## 函式庫 (已內含)  
(1) https://github.com/DFRobot/DFRobotDFPlayerMini
